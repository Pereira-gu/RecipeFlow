# Agents in RecipeFlow

This document outlines the AI agents used in the RecipeFlow project and their respective roles.

## 1. Recipe Curator Agent
- **Role**: Responsible for searching and suggesting recipes based on user preferences and available ingredients.
- **Capabilities**: API integration with recipe databases, ingredient analysis, and dietary restriction filtering.

## 2. Shopping Assistant Agent
- **Role**: Manages the shopping list and optimizes ingredient purchasing.
- **Capabilities**: Converting recipes to shopping lists, price comparison, and inventory management.

## 3. Nutritional Advisor Agent
- **Role**: Provides nutritional information and health-related advice.
- **Capabilities**: Calorie counting, macro-nutrient breakdown, and suggesting healthier alternatives.

## 4. UI/UX Agent
- **Role**: Ensures the mobile application interface is intuitive and accessible.
- **Capabilities**: Layout suggestions, accessibility checks, and theme consistency.
